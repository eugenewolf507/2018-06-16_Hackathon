[Программа курса]

  • кто такой pm?

    Какие качества и навыки важны для ПМа в 2018 году? Классификации проектов по типу и источнику.

  • Waterfall, Lean, Scrum, Kanban как методологии управления

    Fixed price, unit contracts, time and material contacts.

  • Инициация IT-проекта: Lean Canvas, Roadmapping

    Заполняем Топ-5 документов для старта проекта.

  • Roadmap to iterations — user story mapping, user story creation

    Учимся делить стадии проекта на задачи и екомпозируем их.

  • Quality assurance, test plan creation

    Чем управление качеством отличается от тестирования? Какие основные этапы тестирования и зачем это знать ПМу? Создаем тест план и тест кейсы.

  • Работа с командой

    Подбор и поиск A-players Мотивация разных типов сотрудников Лидерство: применение современных концепций на практике

  • Ключевые метрики IT-проекта

    Какие показатели важны для команды, для заказчика, cпонсора и инвестора?

  • Проведение эффективных собраний

    Как правильно организовать и проводить ежедневные, еженедельные и ежемесячные собрания с командой
